2. 动画实现
**定时器版本**
<!-- 效果如下：
<img src="/img/canvas基本绘制.png" alt="图片描述"> -->

思路： 原图包含小人的走路姿势，用一个定时器，每次平移即可实现小人走动的动画效果，当然，注意时间。
```javascript
    const draw = () => {
      const img = new Image()
      img.src = require('./只狼.jpeg')
      img.onload = () => {
        let setup = 0
        setInterval(() => {
          c.clearRect(0, 0, WIDTH, HEIGHT)
          c.drawImage(img, 184 * setup, 0, 184, 325, 0, 0, 184, 325)
          setup++
          setup = setup % 8
        }, 10)
      }
    }
```
